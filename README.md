## Description
A JavaFX-based Sudoku Solver application that allows users to input an unsolved Sudoku puzzle
and automatically solves it using a backtracking algorithm.

## Features
- User-driven input
- GUI-based Sudoku grid
- Automatic puzzle solving
- Clear and simple interface

## Technologies Used
- Java
- JavaFX
- IntelliJ IDEA

## How to Run
1. Open in IntelliJ
2. Add JavaFX library
3. Set VM options:
   --module-path "C:\Program Files\Java\javafx-sdk-21.0.10\lib" --add-modules javafx.controls
4. Run MainApp.java
